Personal Expense Tracker Application

Developed a simplified web-based personal expense tracker facilitating the management of transactions across various categories.

Project Overview

Users can add, record, and manage transaction details, including category management with options to add, update, or delete categories. The application features Date, Amount, and Category filters for efficiently loading transaction data and provides a pie chart representation for percentage-wise comparison of expenses. Users can export transaction table data based on specified duration, category, and amount filters in Excel format. REST API is utilized for effective communication.

Role & Responsibilities

Role: Full Stack Developer

Managed database operations, ensuring secure access and implementing robust mechanisms for storing and retrieving data securely. Implemented filters for Date, Amount, Category for efficient loading of transaction data. Implemented category management functionalities including adding, updating, and deleting categories.
