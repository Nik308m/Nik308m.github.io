Web Based AddressBook

Developed a web-based application for managing contact information, offering seamless storage and retrieval of contact details.

Project Overview

This AddressBook application allows users to store and retrieve contact information such as first name, middle name, last name, email, mobile number, and address. It utilizes a database for data storage and provides search functionality to find contacts based on first or last names.

Role & Responsibilities

Role: Full Stack Developer

Orchestrated end-to-end development, maintenance, and enhancements of both backend and frontend components for the AddressBook application. Managed database operations for secure data storage and retrieval.

Tech Stack

  • Backend Development:
    • Framework: Spring Boot
    • Language: Java (SE 17)
    • Database: PostgreSQL, MySQL, MariaDB
    • API: RESTful API
    • ORM: Hibernate (Spring Data JPA, Plain JDBC)
    • Build Tool: Maven
  • Frontend Development:
    • Languages: HTML, CSS, JavaScript
